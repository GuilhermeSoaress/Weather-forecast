# SkyCast

Modern weather forecast application built with React 19 and Tailwind CSS v4.

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- Zustand
- Axios
- Lucide React
- OpenWeatherMap API

## Getting Started

```bash
npm install
npm run dev
```

## Development

```bash
npm run build
npm run preview
npm run lint
```
